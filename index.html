Paste this into Claude Code, GitHub Copilot, Cursor, or any AI assistant with web access. That's it.

How it works

Four things happen — none of them are your job.

You type one prompt. Your AI assistant does the rest, then hands you a single self-contained HTML file.

1

You give the topics

"Polish news, football, motorization, climbing" — or leave it blank for the default AI/tech coverage.

2

AI reads the playbook

Your assistant fetches this repo's prompts and stylesheet. No fork, no clone.

3

AI does the research

It searches the web, verifies dates and sources, picks the strongest 3-4 stories per section.

4

You get an HTML file

Self-contained. CSS inline. Opens with a double-click. No internet required to view.

🎯 Why this exists

If you're new to AI and you've heard "you can build anything with AI agents now" but every example is wrapped in five layers of frameworks — this repo is for you.

It's a tiny, real, working AI tool you can run, read, modify, and steal from. There is no code to run, no server, no dependencies. Just Markdown and CSS, end to end. If you've ever copy-pasted a Python script, you can make this yours.

What this repo does NOT do

Boring tech, on purpose.

Every "missing" feature below is a deliberate choice that supports the mission.

✗ No Python or Node.js to run
✗ No requirements.txt
✗ No API keys or secrets
✗ No CI / no cron
✗ No GitHub Pages deploy of generated briefs
✗ No database, no server
✓ Plain Markdown agent prompts
✓ Plain HTML + CSS template
✓ Reads end-to-end in 30 minutes

Want to make it yours?

Fork the repo, edit agents/coverage.md for default topics and templates/style.css for the look. Then point your AI at your fork. Done.

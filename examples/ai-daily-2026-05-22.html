Read full brief

  • Self-hosted sandboxes (public beta): the agent loop stays on Anthropic infra, but every tool call — file reads, API hits, code execution — runs inside the customer's environment.
  • Managed providers supported on day one: Cloudflare, Daytona, Modal, Vercel.
  • MCP tunnels (research preview): a lightweight gateway you deploy inside your network opens a single outbound connection. Internal DBs, knowledge bases, ticketing become Claude tools without public endpoints.

The single-outbound-connection design is the part that matters: most enterprise security teams will refuse anything that needs an inbound firewall hole.

Implications: The "agent connectivity layer" Anthropic just expanded (after acquiring Stainless earlier this week) is becoming a durable moat. Expect OpenAI/Google to ship comparable self-hosted sandbox stories within a quarter, but tunnels-first MCP is harder to copy because the protocol is Anthropic-anchored.
